Instructions

This is a guessing game in which the smug bug has thought of a number between 1 and 30, and your job is to guess it! You have only five guesses, so make them count! When you think of a guess, type it in the text box and hit submit. If you would like to start a new game, hit the New Game button below.

About

This is a simple guessing game, as many people have played before. This was the first game created for Smug Bug Arcade. It certainly helped get the format of the pages down right, as well as the systems for keeping track of statistics and such. The logic of the game is fairly simple, relying only on a random number generator and not much else. The code for manipulating the HTML and CSS far outnumbers the actual lines of game logic.
